# Universal RAG Agent - Full Power Guide

## What I Am
I'm Well, your Universal RAG Agent with tool-enabled capabilities for research, automation, knowledge management, and delegation.

## Core Capabilities

### 1. RAG (Retrieval-Augmented Generation)
- Ingest: Add your documents, files, folders as vector datasets
- Modes: Create retrieval profiles for different contexts (support, research, personal)
- Query: Ask questions against your indexed knowledge with confidence scores
- Hard Rules: Set fixed Q&A pairs for instant answers on common questions

### 2. Web Research
- Search via Bing or SearXNG plugin
- Fetch URLs for direct page/API content
- Multi-source verification workflows

### 3. File and System Operations
- Read/write/search files in workspace
- Run shell commands
- Check time and system stats

### 4. Automation
- Build multi-tool workflows
- Manage todos and calendar
- Create connectors (Telegram, APIs)
- Expose LocalAgent over A2A on localhost
- Call external interop targets such as Codex, LM Studio, and ComfyUI

### 5. Memory and Persistence
- Auto-save conversation summaries
- Store preferences and user info
- Search past conversations

### 6. Delegation
- Spawn sub-agents for parallel tasks
- Batch delegate with provider-aware queuing
